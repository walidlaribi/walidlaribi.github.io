Let's geek out!

I was looking at this year’s Google I/O website, please take a look at it. This is the design I really love: Simple, flat, creative and modern! My hope every web designer/developer get inspired by this website in the future for better quality!

If you notice in the second screen of the website, you’ll find a nice photo and some text that is being written continuously on top of it with some cool words: Let’s geek out, Let’s hack, Let’s reboot, Let’s do this, etc.

I personally liked it so much and found it original, so today, we’re going to recreate it with HTML, Javascript and CSS3. This is going to be very simple, even if you don’t have a lot of knowledge in JS or CSS3 you can easily follow the tutorial and understand how it is working.

Let’s start!

First of all, we have to define the structure of our HTML page. We should put a background image, inside it we have a static text (Let’s), next to it a dynamic text (geek out) and finally a blinking cursor. So we will have the following structure:

<section class="background">
	<section class="text">
		<span class="static-word">Let's</span>
		<span id="type-out__word">(dynamic text)</span>
		<span class="type-out__beam blink">&nbsp;</span>
	</section>
</section>

This is our file structure, very simple! Now we need to add a script in order to manipulate this document and add our typing effect to it. I’ll first explain how does it work. First, we will define an array of words that are going to be written in the dynamic text containet span. Then, we will use Javascript timing methods setInterval() and setTimeout() for the delays, so the text will not appear at once, but we’ll simulate the writing instead.

Javascript’s setInterval() method takes two arguments, a callback and a time interval just like this:

setInterval(function() { //do stuff here; }, 1000); //interval in milliseconds

This means that anything you put in the //do stuff here; section will be executed repeatedly every 1 second. If you want to stop this execution, then you’ll have to call the clearInterval() method, by giving the name of your setInterval() method as a parameter just like this:

var myIntervalMethod = setInterval(function() { //do stuff here; }, 1000); //interval in milliseconds
clearInterval(myIntervalMethod);

The other timing method that we’re going to use is the setTimeout() method which works like setInterval() but it executes the callback method only once. It’s like you’re telling your brwoser: Okay, please wait 1000 milliseconds, then execute this callback method.

You can read more about Javascript’s timing methods by following THIS link.

So here is our Javascript code to simulate the typing:

//First, we get our span in which we're going to display dynamic text.
var word_holder = document.getElementById("type-out__word");

//We define our array of words that we want to display, you can add as many words as you want.
var words = ['geek out', 'break stuff', 'reboot', 'hack', 'grab a drink', 'do this', 'create', 'learn', 'invent'];

//word_index is the index in the array of the word being typed.
var word_index = 0;

//char_index is the index in the word of the character being typed.
var char_index = 0;

//we tell the browser to execute writeText() method every 50 milliseconds.
myWriteInterval = setInterval(function() { writeText() }, 50);

//we define writeText() method responsible of typing characters in the dynamic text span.
function writeText() {
	//we add the character in char_index position of the word_index word to our span's innnerHTML.
	word_holder.innerHTML += words[word_index][char_index];
	//increment the char_index to write the next character during the next call of the method
	char_index++;
	//if we've just types the last character of a word, increment word_index and call stopInterval() to wait a little bit.
	if (char_index == words[word_index].length) {
		word_index++;
		char_index = 0;
		stopInterval();
	};
	//if we've just wrote the last word of the array, we start over.
	if (word_index == words.length) {
		word_index = 0;
	};
}

//we define our stopInterval() method responsible of pausing the typing a little bit and then restarting it.
function stopInterval() {
	//clearInterval() stops the actual execution of myWriteInterval function.
	clearInterval(myWriteInterval);

	//here we call setTimeout() in order to wait 3 seconds before calling myWriteInterval again to fire up the typing of the next word.
	setTimeout(function() { 
		//we clear the content of our span.
		word_holder.innerHTML = "";
		myWriteInterval = setInterval(function() { writeText() }, 50); 
	}, 3000);
}

If you execute this code now, it should display “Let’s ” plus a word from the array with typing being simulated and all. This is perfect. But you will notice that it just doesn’t look like Google’s one, ours is ugly! Of course, here is when CSS comes into the game :)

In order for our page to look more beautiful, we will add the following styling to it, nothing special.

<style>
@import url(http://fonts.googleapis.com/css?family=Roboto:100,400);

.type-out__beam{
	display:inline-block;
	margin-bottom:-0.12em;
	width:0.04545em;
	line-height:1;
	min-width:1px;
	background-color:#fee94e;
	vertical-align:baseline
}

.static-word {
	color: white;
}

.background {
	font-family: 'Roboto', sans-serif;
	font-weight:  100;
	font-size: 4.44444em;
	line-height: 1.125;
	color: #fee94e;
	background-image:url("https://www.google.com/events/io/images/photos/experience-1400.jpg");
	height:  500px;
	padding-top: 400px;
	margin: 0;
}

.text {
	text-align: center;
	font-weight: 400;
}
</style>

Put all together and you should have the following result:

You will tell me now that the cursor is not blinking like Google’s one! You are wright, because we haven’t add any CSS3 animations yet. Add the following code into your style:

@-webkit-keyframes blink{
	0%{opacity:1}
	50%{opacity:0}
}
@-moz-keyframes blink{
	0%{opacity:1}
	50%{opacity:0}
}
@-ms-keyframes blink{
	0%{opacity:1}
	50%{opacity:0}
}
@keyframes blink{
	0%{opacity:1}
	50%{opacity:0}
}

.type-out__beam.blink{
	-webkit-animation:blink 1s step-end infinite;
	-moz-animation:blink 1s step-end infinite;
	-o-animation:blink 1s step-end infinite;
	animation:blink 1s step-end infinite
}

This will tell the browser to look for a tag with .blink class and apply the animation with the name blink to it. This animation has been defined above. Basically, it sets the opacity of the object to 1 in 0% keyframe and to 0 in 50% keyframe. That’s all.

Now when you put all of this together you should see a very nice typing with styles and everything, exactly like Google’s one and here is the demo :
